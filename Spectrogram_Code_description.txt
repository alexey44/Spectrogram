
	Spectrogram - описание структуры программы.

Программа реализована в виде оконного приложения. Имеется стандартное меню с различными функциями:
	- работа с файлами;
	- функции цифровой обработки сигнала;
	- функции выбора/изменение кодировки файла;

В программе реализована многовкладочная структура (QTabWidget), что дает возможность анализировать 
несколько файлов одновременно.

Два временных графика представляю собой следующую структуру: 
QLineSeries -> QChart -> QGraphicsScene-> QChartView -> QWidget.

QLineSeries - осуществляет линейную интерполяцию между отсчетами сигнала на графике.
QChart - отображает QLineSeries и оси координат.
QGraphicsScene- необходимо для определения позиции курсора при клике мыши и необходимой обработки после нажатия.

Спектрограмма представляет собой изображение (QImage) заполненное согласно алгоритму, 
который описан в файле Spectrogram_DSP_description.txt

Внизу программного окна расположены редактируемые поля ввода-вывода параметров курсоров.

Описание файлов исходного кода:
mainwindow - реализация многовкладочной структуры, обработка команд меню.
calculatingfft - расчет БПФ и формирование спектрограммы, а также синтез КИХ-фильтров.
first_scene - обработка кликов мыши в области построения временных графиков
spectrewidget - реализация основных функций программы: масштабирование, обработка курсорных измерений,
		работа с кодировкой файла, открытие/считывание/сохранение файлов, реализация всех функций ЦОС
spectrearea - реализация обработки курсоров на спектрограмме.
