
	HowToUse: программа "Spectrogram"

Программа предназначена для визуального анализа и цифровой обработки записей радиосигналов.

Входные данные: упорядоченный бинарный файл с отсчетами сигнала: 1 отсчет представляет собой пару чисел (числа записаны в дополнительном коде) -
 	вещественная и мнимая компоненты отсчета. Например, если один отсчет представляет собой 4 байта, то первые 2 байта - вещественная часть, а вторые 2 байта мнимая часть. 
	Соответственно с 1-го по 4-й байт записан первый отсчет, с 5-го по 8-й байт записан второй отсчет и т.д. 

Визуальный анализ записи сигнала - реализован при помощи построения 2-х временных графиков (верхний исходный сигнал, нижний - увеличенный фрагмент верхнего графика),
и частотно-временного графика, т.е. спектрограммы. 
На временных графиках отображается зависимость от времени значения вещественной (синий) и мнимой (красной) компонент отсчетов сигнала.
На спектрограмме показана зависимость от времени мгновенного амплитудного спектра сигнала: чем темнее участок спектрогрммы, тем выше мощность соответствующих гармоник сигнала.

Визуальный анализ позволяет оценить: наличие/отсутствие сигнала в исследуемой полосе частот, а также
				     оценить параметры сигнала при помощи курсоров (центральную частоту, ширину полосы, длительность фрагмента сигнала).
			
Реализованы следующие функции ЦОС: перенос на нулевую частоту, фильтрация (ФНЧ, ФВЧ), передискретизация.

Дополнительные функциональные возможности:
	- анализ файлов любого размера (на временные графики выводится не более 1000 отсчетов из файла равномерно взятых по всему объему данных)
	- считывание и преобразование записи сигнала в различные типы кодировок.

Сочетания клавиш для работы с ПО "Spectrogram":
	- shift + ЛКМ => установка первого (желтого) курсора 
	- shift + ПКМ => установка второго (зеленого) курсора 
	- ctrl + ЛКМ => установка левой границы области увеличения
	- ctrl + ПКМ => установка правой границы области увеличения
	- средняя кнопка мыши => построение спектрограммы и увеличенного фрагмента сигнала на 2-м графике. (спектрограмма строится для области "увеличенной" области данных).
	- нажатая ЛКМ и выделение интересующего фрагмента графика в направлении слева-направо => увеличение фрагмента данных на временном графике.
	- нажатая ЛКМ и перемещение курсора в направлении справа-налево => возврат к исходному масштабу графика.

Все виды цифровой обработки выполняются в "промежутке" между двумя курсорами:
	- если существуют оба курсора, то обрабатывается фрагмент данных между ними, в случае если желтый курсор "левее" зеленого, иначе (зеленый "левее" желтого) 
	обрабатывается весь файл с удалением области между курсорами.
	- если существует только желтый курсор: происходит обработка, начиная от позиции желтого курсора и до конца файла.
	- если существует только зеленый курсор: происходит обработка, начиная от начала файла до позиции зеленого курсора.
